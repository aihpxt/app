## 🛠️ 技术架构

### 前端技术栈
- **框架**：Vue 3 + Vite
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **HTTP客户端**：Axios
- **可视化**：ECharts
- **图标**：@element-plus/icons-vue

### 后端技术栈
- **AI服务框架**：FastAPI
- **算法库**：NumPy, Pandas, Scikit-learn
- **爬虫**：BeautifulSoup, Requests
- **部署**：Uvicorn

### OpenClaw小龙虾智能体系统
- **LLMService** - 大语言模型服务
- **RAGSystem** - 检索增强生成系统
- **RuleEngine** - 规则引擎
- **UserProfileSystem** - 用户画像系统
- **PolicyCrawler** - 政策爬虫
- **ContentRiskControl** - 内容风险控制
- **OpenClawGateway** - 服务网关

## 🚀 快速开始

### 启动AI服务
```bash
cd ai-service
pip install -r requirements.txt
python main.py
```
服务地址：http://localhost:8000

### 启动前端服务
```bash
cd frontend
npm install
npm run dev
```
服务地址：http://localhost:3002

### 启动后端服务 (可选)
需要配置MySQL和Redis
