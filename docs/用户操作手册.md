## 1. 快速开始

### 1.1 系统要求

- 现代浏览器（Chrome、Firefox、Edge、Safari 最新版本）
- 稳定的网络连接
- JavaScript 已启用

### 1.2 访问系统

在浏览器中输入平台地址即可访问系统首页。

## 2. 注册与登录

### 2.1 注册账号

1. 点击首页右上角的「注册」按钮
2. 填写以下信息：
   - 用户名（3-20个字符）
   - 邮箱（用于验证和找回密码）
   - 密码（至少6位，包含字母和数字）
   - 确认密码
   - 手机号（可选）
3. 阅读并勾选「服务条款」和「隐私政策」
4. 点击「注册」完成注册

### 2.2 登录系统

1. 点击首页右上角的「登录」按钮
2. 输入用户名或邮箱
3. 输入密码
4. 点击「登录」进入系统

### 2.3 忘记密码

1. 点击登录页面的「忘记密码」
2. 输入注册时的邮箱
3. 查收邮件中的重置链接
4. 点击链接设置新密码

## 4. 学校查询

### 4.1 搜索学校

1. 进入「学校查询」页面
2. 使用搜索框输入学校名称或关键词
3. 使用筛选条件：
   - 地区（州市/区县）
   - 学校类型（公办/民办）
   - 录取批次
   - 特色标签
4. 点击「搜索」查看结果

### 4.2 查看学校详情

1. 在搜索结果中点击学校卡片
2. 查看学校信息：
   - 基本信息（名称、地址、电话）
   - 学校简介
   - 历年录取分数线
   - 招生计划
   - 学校特色
   - 校园环境图片

### 4.3 收藏学校

1. 在学校详情页点击「收藏」按钮
2. 收藏的学校可在「个人中心」-「我的收藏」中查看

## 5. AI智能择校

### 5.1 使用AI择校

1. 进入「AI智能择校」页面
2. 填写以下信息：
   - 中考预估分数
   - 所在地区
   - 兴趣方向（理科/文科/艺术/体育）
   - 其他偏好
3. 点击「开始推荐」
4. 查看AI生成的推荐学校列表

### 5.2 推荐结果解读

AI推荐结果包含：
- 学校名称和基本信息
- 录取概率评估（高/中/低）
- 推荐理由
- 志愿填报建议

---

## 6. 分数预测

### 6.1 成绩预测

1. 进入「分数预测」页面
2. 输入历次模拟考试成绩
3. 填写学习情况和目标学校
4. 点击「预测」
5. 查看预测结果和提升建议

---

## 7. 志愿填报

### 7.1 模拟填报

1. 进入「志愿填报」页面
2. 根据中考分数选择合适的学校
3. 按顺序填写志愿表
4. 点击「保存方案」
5. 可保存多个方案进行对比

### 7.2 志愿建议

系统提供志愿填报建议：
- 冲一冲的学校
- 稳一稳的学校
- 保一保的学校

---

## 8. 学校对比

### 8.1 添加对比学校

1. 进入「学校对比」页面
2. 点击「添加学校」
3. 选择要对比的学校（最多5所）
4. 点击「开始对比」

### 8.2 查看对比结果

对比维度包括：
- 历年分数线
- 招生计划
- 学校特色
- 升学情况
- 地理位置
