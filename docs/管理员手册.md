## 1. 管理员账号

### 1.1 账号权限

管理员账号拥有以下权限：
- 用户管理（查看、编辑、禁用用户）
- 学校管理（添加、编辑、删除学校信息）
- 政策管理（发布、更新中考政策）
- 内容管理（管理页面内容）
- 系统监控（查看系统状态）
- 日志审计（查看操作日志）

### 1.2 登录后台

1. 访问系统登录页面
2. 输入管理员账号和密码
3. 点击「登录」
4. 登录成功后可进入后台管理

### 1.3 修改密码

1. 进入个人中心
2. 点击「修改密码」
3. 输入旧密码和新密码
4. 确认新密码
5. 点击「保存」

## 3. 用户管理

### 3.1 查看用户列表

1. 进入「用户管理」页面
2. 查看所有注册用户
3. 支持按以下条件筛选：
   - 注册时间
   - 用户状态（正常/禁用）
   - 用户类型（普通用户/管理员）

### 3.2 查看用户详情

1. 在用户列表中点击用户
2. 查看用户基本信息
3. 查看用户操作记录
4. 查看用户学习进度

### 3.3 编辑用户信息

1. 在用户详情页点击「编辑」
2. 修改用户信息
3. 点击「保存」

### 3.4 禁用/启用用户

1. 在用户列表中选择用户
2. 点击「禁用」或「启用」
3. 确认操作

## 4. 学校管理

### 4.1 添加学校

1. 进入「学校管理」页面
2. 点击「添加学校」
3. 填写学校信息：
   - 基本信息（名称、地址、电话）
   - 学校简介
   - 历年录取分数线
   - 招生计划
   - 学校特色
   - 上传校园图片
4. 点击「保存」

### 4.2 编辑学校信息

1. 在学校列表中点击「编辑」
2. 修改学校信息
3. 点击「保存」

### 4.3 删除学校

1. 在学校列表中勾选要删除的学校
2. 点击「删除」
3. 确认删除操作

### 4.4 批量导入学校

1. 点击「批量导入」
2. 下载导入模板
3. 填写学校信息
4. 上传Excel文件
5. 确认导入

## 9. 备份与恢复

### 9.1 数据库备份

**手动备份：**
```bash
cd ai-service
python backup_db.py
```

**自动备份（通过crontab）：**
```bash
# 每天凌晨2点备份
0 2 * * * cd /path/to/aiphxt-app/ai-service && python backup_db.py
```

### 9.2 查看备份列表

```bash
python backup_db.py list
```

### 9.3 恢复备份

```bash
python backup_db.py restore backups/schools_backup_xxx.db
```

### 9.4 文件备份

定期备份以下目录：
- `ai-service/data/` - 数据文件
- `ai-service/uploads/` - 上传文件
- `frontend/dist/` - 前端构建文件

## 附录

### A. 环境变量配置

重要环境变量：
```bash
# 服务器配置
HOST=0.0.0.0
PORT=8001
DEBUG=false

# 数据库配置
DATABASE_URL=sqlite:///data/schools.db

# Redis配置
REDIS_URL=redis://localhost:6379/0

# 安全配置
SECRET_KEY=your-super-secret-key
ALGORITHM=HS256

# CORS配置
ALLOWED_ORIGINS=https://your-domain.com

# HTTPS配置
HTTPS_ENABLED=true
SSL_CERT_FILE=certs/server.crt
SSL_KEY_FILE=certs/server.key

# DeepSeek配置
DEEPSEEK_API_KEY=your-deepseek-api-key
DEEPSEEK_API_URL=https://api.deepseek.com/v1
```

### B. 常用命令

**启动服务：**
```bash
cd ai-service
python start_service.py
```

**停止服务：**
```bash
# 查找进程
ps aux | grep uvicorn
# 停止进程
kill <pid>
```

**查看日志：**
```bash
cd ai-service
tail -f logs/app.log
```

**健康检查：**
```bash
curl http://localhost:8001/health
curl http://localhost:8001/health/detailed
```
