## 软件技术特点

### 软件用途
本软件是一个面向云南全省初三考生、家长、学校及教育管理部门的智能决策平台，提供分数核验、录取概率预测、志愿智能推荐、学校全景对比、政策智能解读、数据可视化分析等全链路服务。

### 主要功能
1. **用户管理**：注册登录、个人信息管理、考生信息录入与核验
2. **学校查询**：学校信息检索、详情展示、录取数据查询
3. **AI择校决策**：录取概率预测、志愿智能推荐、分数校准
4. **政策服务**：政策智能解读、查询与筛选
5. **数据统计**：数据可视化分析、热门学校排名
6. **智能体系统**：多智能体协作，提供个性化服务
7. **爬虫系统**：自动爬取政策和学校数据，保持数据更新

### 技术架构
- **前端**：Vue 3 + Vite + Element Plus
- **后端**：Spring Boot 3.2.x + Spring Data JPA + Redis
- **AI服务**：FastAPI + NumPy + Pandas + Scikit-learn

### 技术创新点
1. **AI赋能**：利用人工智能技术，为考生提供智能择校决策支持
2. **多智能体系统**：采用多智能体协作模式，提高服务质量
3. **智能爬虫**：自动爬取政策和学校数据，保持数据更新
4. **数据可视化**：直观展示中考相关数据，帮助用户理解趋势
5. **个性化推荐**：基于考生情况，提供个性化的学校推荐
