## 2. 软件功能概述

云南省AI全域赋能中考择校智能决策平台是一个面向云南全省初三考生、家长、学校及教育管理部门的智能决策平台，提供分数核验、录取概率预测、志愿智能推荐、学校全景对比、政策智能解读、数据可视化分析等全链路服务。

## 3. 软件技术架构

### 3.1 前端技术栈
- **框架**：Vue 3 + Vite
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **HTTP客户端**：Axios
- **可视化**：ECharts

### 3.2 后端技术栈
- **框架**：Spring Boot 3.2.x
- **数据库**：SQLite
- **ORM**：Spring Data JPA
- **安全**：Spring Security + JWT

### 3.3 AI服务技术栈
- **框架**：FastAPI
- **算法库**：NumPy, Pandas, Scikit-learn
- **部署**：Docker

## 6. 软件核心功能说明

### 6.1 智能择校决策
基于AI技术，结合考生分数、排名、兴趣爱好等因素，为考生推荐最适合的学校。

### 6.2 录取概率预测
根据历史录取数据和当前考生情况，预测考生被目标学校录取的概率。

### 6.3 政策智能解读
利用NLP技术，对中考政策进行智能解读，帮助考生和家长理解政策内容。

### 6.4 学校全景对比
提供多维度的学校对比功能，包括师资、设施、升学率等方面。

### 6.5 数据可视化分析
通过图表展示中考相关数据，帮助用户直观了解趋势和变化。
