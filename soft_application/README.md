## 软著申请流程

1. **准备材料**：
   - 填写软著申请表
   - 准备软件说明书
   - 准备源代码（前后端和AI服务核心代码）

2. **提交申请**：
   - 登录中国版权保护中心网站
   - 注册账号并登录
   - 填写软件著作权登记申请表
   - 上传准备好的材料
   - 缴纳申请费用

3. **审核**：
   - 等待版权保护中心审核
   - 如有需要，补充材料

4. **领取证书**：
   - 审核通过后，领取软件著作权登记证书

## 注意事项

1. **代码要求**：
   - 源代码应包含软件的核心功能实现
   - 代码应具有可读性和完整性
   - 代码量应符合软著申请要求（一般不少于3000行）

2. **说明书要求**：
   - 说明书应详细介绍软件的功能、技术架构、核心模块等
   - 说明书应包含软件的运行环境、部署方式等信息
   - 说明书应包含软件的技术创新点

3. **申请表要求**：
   - 申请表应填写完整、准确
   - 软件名称应与实际软件名称一致
   - 版本号应与实际软件版本一致

4. **其他要求**：
   - 所有材料应使用中文填写
   - 所有材料应保持清晰、整洁
   - 所有材料应符合版权保护中心的要求
