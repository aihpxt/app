# 小龙虾择校平台商业模式与收费标准

## 一、平台定位

**小龙虾择校**是云南省专业的中考择校智能决策平台，为考生和家长提供：
- AI智能择校咨询
- 分数预测与志愿填报
- 学校数据查询与对比
- 政策解读与指导

---

## 二、商业模式

### 1. 会员订阅模式（核心收入）

采用**Freemium（免费增值）**模式，通过免费功能吸引用户，高级功能付费解锁。

### 2. 收入来源

| 收入类型 | 说明 | 占比预估 |
|---------|------|---------|
| 会员订阅 | 月/季/年费会员 | 60% |
| 增值服务 | 专家一对一咨询 | 25% |
| 广告合作 | 教育机构推广 | 10% |
| 数据服务 | 学校数据报告 | 5% |

## 四、定价策略

### 1. 价格锚定
- 免费版作为入门，降低使用门槛
- VIP版作为高端锚点，衬托高级版性价比

### 2. 促销策略
| 活动 | 内容 | 时间 |
|-----|------|-----|
| 新用户优惠 | 首月5折 | 注册后7天内 |
| 年度大促 | 年付8折 | 每年6-8月 |
| 老用户续费 | 续费9折 | 到期前30天 |
| 推荐奖励 | 推荐返现20% | 长期 |

### 3. 差异化定价
- **学生优惠**：凭学生证享8折
- **教师优惠**：凭教师资格证享7折
- **团购优惠**：3人以上团购享85折

## 五、用户增长策略

### 1. 获客渠道
- 微信公众号内容营销
- 短视频平台（抖音、快手）
- 学校合作推广
- 口碑传播（推荐奖励）

### 2. 转化漏斗
```
访问用户 → 注册用户 → 免费体验 → 付费转化 → 续费复购
   100%      30%        20%        10%        60%
```

### 3. 留存策略
- 每日签到送次数
- 积分兑换权益
- 会员专属活动
- 定期推送有价值内容
